📊 Data Cleaning & SQL Table Creation Project

This project was completed as part of my Data Analytics Internship. The main objective of this project is to understand how to create tables using SQL and perform data cleaning and preparation using Jupyter Notebook.

🚀 Project Overview

In this project, raw data is processed and cleaned to make it ready for analysis. SQL is used to create structured tables, and Jupyter Notebook is used for data preprocessing and cleaning tasks.

🛠 Technologies Used

Jupyter Notebook

Python

SQL

📂 Project Tasks

Creating database tables using SQL

Importing and handling datasets

Data cleaning and preprocessing

Removing missing or duplicate values

Preparing structured data for analysis

🎯 Purpose

The purpose of this project is to gain practical experience in database table creation, data cleaning, and data preparation, which are important steps in the data analytics workflow.
